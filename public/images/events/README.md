# Event Images Requirements

## Image Specifications

1. **Hero Banner** (`hero-banner.jpg`)
   - Dimensions: 1920x1080 pixels
   - Format: JPG
   - Content: A wide shot of a food truck event with multiple trucks and people

2. **Northern Food Fest**
   - Main image (`northern-fest.jpg`): 800x600 pixels
   - Additional images:
     - `northern-fest-1.jpg`: 800x600 pixels
     - `northern-fest-2.jpg`: 800x600 pixels
     - `northern-fest-3.jpg`: 800x600 pixels
   - Content: Images of the Northern Food Fest event

3. **Summer Street Food Market**
   - Main image (`summer-market.jpg`): 800x600 pixels
   - Additional images:
     - `summer-market-1.jpg`: 800x600 pixels
     - `summer-market-2.jpg`: 800x600 pixels
     - `summer-market-3.jpg`: 800x600 pixels
   - Content: Images of the summer market event

4. **Food Truck Festival**
   - Main image (`food-truck-fest.jpg`): 800x600 pixels
   - Additional images:
     - `food-truck-fest-1.jpg`: 800x600 pixels
     - `food-truck-fest-2.jpg`: 800x600 pixels
     - `food-truck-fest-3.jpg`: 800x600 pixels
   - Content: Images of the food truck festival

## Image Guidelines
- All images should be high quality and well-lit
- Focus on food, atmosphere, and people enjoying the events
- Maintain consistent aspect ratios
- Optimize images for web (compress without losing quality)
- Ensure good contrast and visibility of food and people
